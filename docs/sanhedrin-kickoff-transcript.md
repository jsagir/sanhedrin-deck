# Sanhedrin Kickoff: Experience and Ideation
## Meeting Transcript - Daniel Muller, Jonathan Sagir, Doron

### Key Decisions and Insights

**1. Machloket vs Debate - RESOLVED**
- Two distinct modes on one platform, not a conflict
- Mode 1: "Culture of Disagreement" (Machloket) - pedagogical, process-focused, collaborative
- Mode 2: "Debate" - competitive, rhetorical, outcome-focused
- Same technology, different protocols/fine-tuning
- This resolves the ongoing tension between Yoni's debate vision and the pedagogical team

**2. Role-Based Architecture (Critical Insight)**
- Separate PERSONA from ROLE
- Roles: Judge (Dayan), Questioner, Supporter, Advocate (Ba'al Din), Synthesizer, etc.
- Any persona (Ben-Gurion, Golda, Rambam) can fill any role
- Humans can also fill roles - swap between human and AI seamlessly
- Based on Niri Dayan's curriculum framework from Beit Sefer L'Tarbut HaMachloket

**3. Human Facilitator is NON-NEGOTIABLE**
- Like a Game Master in D&D / game show host
- Controls the orchestration, has backend dashboard
- Human charisma cannot be replaced by AI
- Facilitators = trained guides from Beit Sefer L'Tarbut HaMachloket

**4. Spectators as Active Judges**
- Not passive watchers
- Can vote, react, provide input via peripheral screens
- Yoni's vision: all the unchosen avatars sit "in the audience"
- Context agent can trigger interruptions from audience avatars

**5. Scenario Design**
- 20+ scenarios from Talmudic sources, modernized
- Key example: "The Goring Ox" -> Autonomous Vehicle liability
- Deliberately non-political topics (moral/ethical gray zones)
- Optimized to give every participant something meaningful to say

**6. The EdTech Play / Debate Gym**
- Museum = premium peak experience
- Schools = structured programs via Beit Sefer L'Tarbut HaMachloket
- Home/Online = open camera, speak, get AI analysis on rhetoric
- Prep tools: learn the rules before arriving, mini-engagement
- Follow-up: students can return, track improvement metrics

**7. What's Solved vs What's Ahead (70/30)**
- Evolution from Rambam: "Rambam was a monologue, Sanhedrin is multi-agent dialogue"
- The EDGE is in the CONNECTION between agentic systems and semi-immersive experiences
- Both exist separately; the innovation is combining them in a new way
- This is what investors need to understand for confidence

**8. Backend Architecture**
- Front-facing agents: avatars with roles (user-visible)
- Orchestrator agent: controlled by facilitator (user doesn't see)
- Background agents: context engine, fact-checker, sentiment analysis
- All running in parallel during the session

**9. Physical Setup**
- Round space, like a game show
- Two groups facing each other (panel style)
- Central facilitator
- Shared immersive display (dashboard updates live)
- Peripheral screens for spectators
- VIP ticketed sessions (like Louis Vuitton museum VR sector)

**10. Next Steps from Meeting**
- Follow-up meeting: Monday at 10am
- Build the concept document and presentation
- Document serves triple purpose: fundraising, board presentation, RFP
- Jonathan to process transcript with notetaker
- Daniel investigating Phantom bot / Tymo Ice for Hebrew AI
